QwikWin (library)

C#

A library using two other custom libraries that provides a performant and highly versatile alternative to Windows Forms. The library provides classes for creating windows and visual components, and makes heavy use of the Windows C/C++ API for the window itself. It utilizes a custom math and vector library, as well as a high performance multithreadable Bitmap wrapper with custom drawing functions which run faster than the built-in methods. QwikWin offers a highly customizable, versatile, and extensible user interface which provides many event handlers and handles many other complex tasks while remaining somewhat similar to using Windows Forms where all visual components share a range of properties, events, and methods in addition to their own as well as a "place and forget" workflow where QwikWin only requires you to create and add components once and it will handle processing and drawing them.

Flashcards

Java

A flashcard program I made to help me study. It shows the front of the card and when clicked, it shows the back along with confidence choices. You answer how well you think you know the back side given the front. Lower confidence answers will make the card have a higher probability of being shown again and answering with full confidence will remove the card from the queue. The blue bar at the top is the progress bar. It is based off of the confidence answers for all the cards in the deck. The bar under that tracks how many cards there are in each confidence category.

Calculator

C#

A calculator that updates as you type. It optionally shows work, supports numbers, vectors, binary, hexadecimal, and scientific notation, can output as a normal number, fraction, scientific notation, hexadecimal, and binary, and has a wide array of functions such as summation, rounding, clamp, and more. It also accepts certain constants like pi and e. The user can assign numbers or vectors to variables too. Internally, numbers are represented by fractions and can be extremely large or small without having floating point precision loss. Nearly all functions work on both vectors and numbers. On vectors, the function is applied to each component. The app is available on GitHub. There is also a library version for use in other projects where the implementing code can just have it try to calculate a number value from a string which is avalable here.
